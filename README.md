## Descrição do Sistema:
Este projeto consiste em um sistema de caixa eletrônico (ATM) desenvolvido em linguagem C. O objetivo é simular algumas operações bancárias básicas, como consulta de saldo, saque e depósito, utilizando estruturas de repetição, switch-case, funções e validação de dados para praticar conceitos fundamentais de programação.


## Funcionalidades do Programa:
- Consulta de saldo
- Realização de saques
- Realização de depósitos
- Menu interativo no console
- Validação de valores e opções do usuário
- Uso de funções para modularização do sistema


## Como Executar no Windows:

Abra o terminal na pasta do projeto e execute o comando abaixo para compilar o programa:

```bash
gcc main.c -o atm
```

Após a compilação, execute:

```bash
atm.exe
```

É necessário ter o GCC/MinGW instalado e configurado no PATH do Windows.
